Grid 레이아웃과 시맨틱 태그 예제

메인 콘텐츠

이 페이지는 CSS Grid 레이아웃과 시맨틱 HTML 태그를 사용하여 만든 반응형 웹페이지입니다. 헤더와 푸터는 항상 전체 너비를 차지하며, 메인 콘텐츠와 사이드바는 화면 크기에 따라 레이아웃이 변경됩니다.

데스크톱 화면(768px 초과)에서는 메인 콘텐츠와 사이드바가 좌우로 나란히 배치됩니다. 메인 콘텐츠는 유연한 너비를 가지며, 사이드바는 300px의 고정된 너비를 유지합니다.

태블릿이나 모바일 화면(768px 이하)에서는 메인 콘텐츠와 사이드바가 세로로 쌓이게 됩니다. 이를 통해 작은 화면에서도 콘텐츠를 쉽게 읽을 수 있습니다.

브라우저 창의 크기를 조절해보면 레이아웃이 자동으로 변경되는 것을 확인할 수 있습니다. 이것이 바로 반응형 웹 디자인의 핵심입니다.
