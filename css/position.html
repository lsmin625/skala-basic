position 데모 (스크롤하면서 fixed / sticky 확인)
1) static (기본값: top/left 적용 안 됨)
static 박스
static은 문서 흐름(normal flow) 그대로 배치되고, top/left/right/bottom은 무시됩니다.
2) relative (원래 위치 기준으로 이동, 자리(공간)는 유지)
relative 박스
relative는 원래 자리에서 이동만 하고, 원래 자리 공간은 그대로 남습니다.
3) absolute (가장 가까운 position 조상 기준으로 배치)
이 컨테이너는 position: relative라서 absolute의 기준점이 됩니다.
absolute 박스
absolute는 문서 흐름에서 빠지고, 기준(조상)의 top/right/left/bottom 기준으로 배치됩니다.
4) sticky (스크롤하다가 top에 달라붙음, 부모 영역 안에서만)
sticky 박스 (스크롤 시 상단에 붙음)
아래 내용을 스크롤해보세요. sticky 박스가 top:0 위치에 달라붙습니다.
sticky는 평소엔 normal flow처럼 있다가, 스크롤로 임계점에 도달하면 고정처럼 동작합니다.
