CSS Position 속성 예제

Static (기본값)
일반적인 문서 흐름에 따라 배치됩니다. top, right, bottom, left 속성이 적용되지 않습니다.
Sticky
스크롤 시 상단에 고정됩니다. (스크롤을 내려보세요!)
Relative
원래 위치를 기준으로 top: 30px, left: 50px 이동했습니다.

이 컨테이너는 absolute 요소의 기준이 됩니다 (position: relative)

Absolute
부모 요소를 기준으로 top: 100px, right: 50px 위치에 배치됩니다.
Fixed
화면 우측 하단에 고정

스크롤을 내려서 Sticky와 Fixed 요소의 동작을 확인하세요!
